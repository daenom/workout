# Workout

Workout tracking and planning application

## Project Structure

- `backend/` - Spring Boot backend application
- `frontend/` - planned frontend application

## Getting Started

### Prerequisites

- Java 25

### Installation

1. Clone the repository

```bash
git clone https://github.com/daenom/workout.git
cd workout
```

2. Run backend

```bash
cd backend
./mvnw spring-boot:run
```
